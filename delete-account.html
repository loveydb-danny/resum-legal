2. 계정 삭제 (앱에서 직접)

앱 내 [설정] > [탈퇴하기]를 통해 계정을 삭제할 수 있습니다.

  1. 리썸 re:sum 앱 실행
  2. 하단 내비게이션 바에서 'MY' 탭 선택
  3. 우측 상단 설정(⚙) 버튼 탭 → 설정 화면 진입
  4. 화면 하단 '탈퇴하기' 버튼 탭
  5. 안내 사항 확인 후 '탈퇴하기' → 확인 팝업에서 '탈퇴하기' 버튼을 눌러 최종 확인
